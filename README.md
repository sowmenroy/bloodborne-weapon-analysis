# Bloodborne Weapon & Boss Analysis

A relational database and analysis project exploring Bloodborne's weapons and boss damage weaknesses, built as a data analyst portfolio project.

## Project Status

| Stage | Status |
|---|---|
| Data collection (CSV) | 🚧 In progress |
| Python cleaning/EDA | ⬜ Planned |
| SQL relational queries | ⬜ Planned |
| Power BI dashboard | ⬜ Planned |

## Project Overview

This project builds a small relational database of Bloodborne weapons and bosses to answer:

**Which damage type (physical, blood, arcane, fire) has the best average coverage against bosses?**

## Planned Structure

- `/data` — raw and cleaned CSV files (weapons, bosses, boss-weapon effectiveness)
- `/python` — data cleaning and exploratory analysis
- `/sql` — schema and queries for relational analysis
- `/powerbi` — dashboard file and visuals

## Tech Stack

- Python (pandas)
- SQL (PostgreSQL)
- Power BI
